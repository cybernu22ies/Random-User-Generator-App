# User Generator App

Простое приложениe для генерации случайных пользователей и отображения их в виде списка. Для генерации данных пользователей используется API <https://randomuser.me/>, удачно сгенерированные пользователи кэшируются на устройстве.

![app screenshots](https://github.com/user-attachments/assets/fd226c9b-77ec-4d69-a938-a732f750d86a)

### Технологии использованные при разработке:
- Jeptack Compose (отрисовка пользовательского интерфейса)
- Room (хранение полученных данных в локальном хранилище на устройстве)
- OkHttp (запросы к API)
- Coil (загрузка и кэширование изображений)
- Hilt (реализация паттерна инъекции зависимостей)
